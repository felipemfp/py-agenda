# py-agenda

*Meu primeiro contato com Python.*

Aplicação de linha de comando para organizar uma Agenda de contatos.

## Como usar?

```shell
git clone https://github.com/felipemfp/py-agenda.git
cd py-agenda
python program.py
```

## O que é necessário?

Você precisa ter instalado o Python 3.4 ou superior.
